# Contributing Notes

공개 저장소에는 public sample, 실행기, 문서, 공개 안전 리더보드 payload만 둔다.
비공개 holdout, 원시 실행 로그, 로컬 handoff 파일은 커밋하지 않는다.

## 문항 추가 시 권장 형식

각 문항은 가능하면 다음 정보를 포함한다.

- ID
- 분야: 회계 / 법인세 / 소득세 / 부가가치세 / 국세기본법 / 기타
- 유형: 객관식 / 계산형 / 사례형 / 근거제시형 / 리스크분석형 / 에이전트형
- 난이도
- 문제 본문
- 정답
- 해설
- 근거 자료
- 채점 루브릭
- 출처 및 저작권 상태
- 기준 시점

## 주의

기출문제 원문, 유료 해설, 교재 내용을 그대로 복제하지 않는다. 초기에는 구조·아이디어·메타데이터 중심으로 정리한다.
