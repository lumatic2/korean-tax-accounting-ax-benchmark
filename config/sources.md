모든 소스는 [data-strategy.md](../docs/data-strategy.md) §2 저작권 지위와 함께 본다. 상태: v0.1 (2026-06-01).

## 인용 형식

- 결과 인용 시: *소스명 + 조문/사건번호 + 핵심 내용 한 줄 + 확인 시점(기준일)* 박제.
- 예: `부가가치세법 제30조(2026 기준) → 일반과세 세율 10% [국세법령정보시스템, 2026-06-01 확인]`
- 예: `대법원 2020두12345 → (쟁점 한 줄) [대법원 종합법률정보, 2026-06-01 확인]`
- 온라인 출처는 가능하면 하이퍼링크(공공누리 출처표시 요건).

## ⚠ 주의

- 모든 외부 호출 결과에 **확인 시점(기준일)** 기록 — 세법은 시점 종속(적시성).
- 캐시된 결과를 *현재 사실*로 다시 인용하지 말 것 (개정 가능성). v2025/v2026 버전 구분.
- **Open API 이용약관**은 저작권(§7)과 별개로 재배포·상업이용을 계약상 제한할 수 있음 → 상업화 전 [확인].
- 예규·질의회신의 §7 해당 여부 미확정 → 인용 시 "[검증 필요]" 표시.
